Vesuvius

Data from outer space

Vesuvius is a small but powerful pipeline that allows any user to query NASA's images of the SpaceX-built DSCOVER satellite, like the one seen here, and enrich an existing dataset of relevant dates.

While the name is a rendition to the legendary Vesuvius volcano located in Italy, our Command Line Interface program uses a dataset of volcanic activity to find available satellite images of these volcanic events given an specific timeframe, and automatically generates a PDF report that is sent by email to any stakeholder.
